# Random Philosophical Quote Generator App

 Are you ready to expand your mind with some inspiring quotes from some of the greatest thinkers in history?
Looking to improve your life with timeless wisdom? This app is for you! So go ahead, download our repository, and discover the quote that will change your life today!

 Simply click the "Generate Quote" button and you'll get a random quote along with the philosophical current of thought of the philosopher who said it. 

 ### Description

This is a web application that generates a random quote from a famous philosopher along with their philosophical current of thought. Simply click the "Generate Quote" button to get started.


### Technologies Used

- HTML
- CSS
- JavaScript

### How to Use

1. Clone this repository or download the ZIP file.
2. Open the `index.html` file in your web browser.
3. Click the "Generate Quote" button to generate a new random quote.

### Sample Quote Output

> "The only true wisdom is in knowing you know nothing." - Socrates (Classical Greek philosophy)
